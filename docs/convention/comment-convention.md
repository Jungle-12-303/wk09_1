# 주석 컨벤션

> 목적: 원본 주석의 의미를 보존하면서, 이후 AI 주석 정리나 컨벤션 적용 과정에서 중요한 주석이 임의로 바뀌지 않도록 한다.

---

## 원본 영어 주석 번역

최초 기준의 영어 주석은 원문의 의미가 누락되지 않도록 한글로 번역한다.

번역이 완료된 주석에는 `@lock` 태그를 추가한다.

`@lock` 태그가 붙은 주석은 이후 AI 주석 정리, 리팩터링, 컨벤션 적용 과정에서 수정하거나 삭제하지 않는다.

```c
/* @lock
 * 타이머 인터럽트 핸들러.
 */
static void
timer_interrupt (struct intr_frame *args UNUSED) {
	...
}
```

---

## 추가 주석 작성

새로 추가하는 주석은 한글로 작성한다.

함수 외부에서 함수에 대한 추가 주석이 필요하면 `@lock` 주석 아래에 작성한다.

```c
/* @lock
 * TICKS 타이머 틱 동안 현재 스레드의 실행을 중단한다.
 */
/* ticks가 0 이하이면 별도 대기 없이 즉시 반환한다. */
void
timer_sleep (int64_t ticks) {
	...
}
```

---

## 주석 위치

주석은 코드 옆이 아니라 코드 위에 작성한다.

```c
/* 현재 스레드가 깨어날 절대 tick을 저장한다. */
curr->wakeup_tick = wakeup_tick;
```

아래처럼 코드 옆에 붙이지 않는다.

```c
curr->wakeup_tick = wakeup_tick; /* 현재 스레드가 깨어날 절대 tick */
```

---

## 형식

여러 줄 주석은 아래 형식을 사용한다.

```c
/* @lock
 * 원문을 빠뜨리지 않고 번역한 주석 본문.
 */
```

`@lock`이 아닌 추가 주석은 아래 형식을 사용한다.

```c
/* 컨벤션에 따라 새로 작성한 주석 본문. */
```

---

## 금지 사항

- `@lock` 주석을 삭제하지 않는다.
- `@lock` 주석의 의미를 바꾸지 않는다.
- 영어 원문 주석을 요약 번역하지 않는다.
- 코드 옆에 새 주석을 붙이지 않는다.
- 함수 외부 설명 주석을 `@lock` 주석보다 위에 두지 않는다.
